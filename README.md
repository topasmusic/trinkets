# Trinkets

Maintained Trinkets fork with two version lines.

- `1.21.11/` is the legacy line.
- `26.1.2/` is the current Minecraft `26.1.2` line.

This repository is based on the original Trinkets mod by Emily Ploszaj and keeps each supported Minecraft line in its own top-level folder. Build and run the version folder you actually want to work on.

Release artifacts are versioned per Minecraft target, for example `26.1.2-trinkets-1.0.jar`.
